# Blinkit Sales Analysis Dashboard

## Project Overview

This project is an interactive Blinkit Sales Analysis Dashboard developed using Power BI.
The dashboard provides detailed insights into sales performance, outlet analysis, product distribution, customer ratings,
and inventory trends across different outlet types and locations.

The main objective of this project is to help businesses analyze sales patterns, monitor outlet performance,
identify high-performing product categories, and support data-driven business decisions using interactive visualizations and KPI tracking.

--------------------------------------------------

## Business Problem

Retail businesses like Blinkit generate huge amounts of sales and inventory data daily. Without proper analysis, it becomes difficult to:

* Monitor overall sales performance  
* Identify top-performing product categories  
* Analyze outlet-wise sales trends  
* Track customer ratings and item visibility  
* Compare sales across outlet sizes and locations  
* Understand inventory distribution across outlets  
* Improve operational and business strategies  

This dashboard solves these challenges through clear visualizations and KPI-based analysis.

--------------------------------------------------

## Tools & Technologies Used

* Power BI — Dashboard Development  
* Power Query — Data Cleaning & Transformation  
* DAX — KPI Calculations  
* Excel / CSV — Dataset Management  
* GitHub — Project Hosting  

--------------------------------------------------

## Dashboard Features


The dashboard provides a complete overview of business performance including:

* Total Sales Analysis  
* Average Sales Tracking  
* Total Number of Items  
* Average Customer Rating  
* Outlet Establishment Trends  
* Sales by Item Type  
* Fat Content Analysis  
* Outlet Size Distribution  
* Outlet Location Analysis  
* Outlet Type Performance Comparison  

--------------------------------------------------

## KPI Metrics Used

* Total Sales — Overall revenue generated  
* Average Sales — Average sales value  
* Number of Items — Total products/items sold  
* Average Rating — Customer satisfaction rating  
* Item Visibility — Product visibility performance  

--------------------------------------------------


## Visualizations Included

--------------------------------------------------

## Conclusion

This Blinkit Sales Analysis Dashboard helps businesses understand sales behavior, outlet performance, customer preferences,
